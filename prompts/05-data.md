# 5. Data

**Localização no trabalho:** Seção 3.1.3 — Data

## O que este prompt gera
A discussão sobre a data de composição do livro (ou, quando aplicável, da perícope), apresentando as diferentes propostas de datação encontradas nas fontes, os argumentos de cada estudioso e uma conclusão sobre o intervalo mais provável.

## Fontes recomendadas para esta seção
Introduções ao Antigo/Novo Testamento, comentários bíblicos, obras de história bíblica.

## Prompt (copie e cole no NotebookLM)

"""

1. O período histórico ou os reinados/eventos que o próprio texto usa para se situar cronologicamente (se houver indicação explícita no primeiro versículo ou em outro ponto do livro).
2. As diferentes datas ou intervalos de datação propostos pelos autores citados nas fontes disponíveis, identificando claramente qual autor propõe qual data e com base em que argumento (cálculo de reinados, eventos históricos externos, evidências internas do texto).

Requisitos de formato:
- Baseie a seção em, no mínimo, 7 autores/obras diferentes das fontes disponíveis, distribuindo as citações entre eles — mais de um autor pode dizer coisas parecidas ou complementares, isso é desejável, pois dá mais opções para você escolher e remover manualmente na revisão final, sem esvaziar a seção. Se as fontes carregadas não sustentarem tantas vozes diferentes para este tópico específico, utilize o máximo de autores distintos disponível e diga isso explicitamente.

## Observações
